# CENTER

## Name
   center [0] -- canhge global map center to another star

## Syntax
    center <star>

## Description
 Global map centerpoint is allways the same system where scope has been
latest. If you like to examine better yhe neigbourghood of some unexplored 
system,  you can change global map centerpoint using center command.
