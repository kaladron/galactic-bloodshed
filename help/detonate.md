# DETONATE

## Name
   detonate [0] -- force a mine to explode

## Syntax
   detonate <mine>

## Description
  Normally, a mine is set to detonate at a prearranged trigger radius
using the 'order' command. However, players may find it useful to order
their space mines to explode upon request, between updates. This
command is used to do this.
