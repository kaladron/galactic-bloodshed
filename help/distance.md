# DISTANCE

## Name
   distance [0] -- get the distance between two objects

## Syntax
   distance <from> <to>

## Description
  Players can use this command to obtain the distances between any pair of
stars/planets/ships.

Example:	dist #123 #456		  give you distance between ships
						#123 and 456
		distance /Sol/Earth /Sol  gives the distance between the star
						and the planet

		distance .Earth #456	  gives the distance between the
						planet and the ship (from
						planet scope.
