# EMOTE

## Name
   emote [0] -- tell all players logged in what are you thinking
		 using third person.	

## Syntax
    emote <message>

## Description
   This command is like broadcast. Text following the emote command 
is sent to each player currently logged in, unless they are GAGed 
(see the toggle command).

## Example

 Player named Skywalker emotes:

'emote just loves Dreadnoughts.'

And every player logged in and ungaged receives message

'Skywalker just loves Dreadnoughts.'
