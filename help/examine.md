# EXAMINE

## Name
   examine [0] -- look at a ship/object

## Syntax
   examine <#shipnum>

## Description
   Examine gives the user a short paragraph-summary description of a ship or
object and how it can be used.  This command can be applied to any ship or
object in a system where a player has equipment (ships, objects, or a 
colonized planet.)
