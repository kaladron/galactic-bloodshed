# JETTISON

## Name
   jettison [0] -- unload commodities from a ship into space

## Syntax
   jettison <#shipnum> <commodity> <amount>

## Description
  The jettison command is used to eject unwanted cargo from ships.

  The permissible commodities are d, c, m, r, f and x meaning destruct,
crew, military, resource, fuel and crystal.

  If <amount> is omitted or zero, all of the commodity is jettisoned.

  Jettison can only be used on ships in space; landed ships cannot
jettison.  Jettisonned cargo is lost forever.
