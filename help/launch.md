# LAUNCH

## Name
   launch [0] -- launch a ship

## Syntax
   launch <#shipnum>

## Description
  This command attempts to launch ships that are landed on planets.  

  To do so a ship must have an amount of fuel proportional to the
gravitational field of the planet and the mass of the ship.  If the ship does
not have enough fuel, it will not be able to launch.  The ship must
not be irradiated.
