# NAME

## Name
   name [0] -- rename an object (ship, block, planet, your race)

## Syntax
   name <what> <name>

## Description
  In order to rename a ship, to the ship, then do 'name ship <name>'.
You can also rename your race with 'name race <name>', or your own governor
name with 'name governor <name>'.  You must own ships to rename them.
You can also name alliance block you lead with 'name block <name>'.
  To name the class of ships being produced by a factory, you use 'name class
<name>'. The factory must be off-line for this to work.
