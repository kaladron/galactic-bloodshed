# PROFILE

## Name
   profile [0] -- racial profile

## Syntax
   profile <race name>

## Description
 This prints out some major information about a race. For example:

race's morale
race's tech (and list of your race's discoveries)
gun ranges
space telescope range
ground telescope range
major sectortype.

 And a bunch of less impotrant info, like capital ship, number of active
updates. This command also prints out a short summary of race's major 
characteristics, like IQ, mass, fight, metabolism...
 You can see race's optimal planet conditions and your own sector
preferences too.
 Add some personal information with 'personal' command.
