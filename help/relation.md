# RELATION

## Name
   relation [0] -- state relations 

## Syntax
   relation <race>

## Description
  This command prints out the relations between the player's race and those
of the other races in the game.  It also gives the translation modifiers
for each race.

  If an argument is given, the relationships towards that race is displayed 
only. If no arguments are given, the relationships towards your race are
shown.
