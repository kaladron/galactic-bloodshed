# SEND

## Name
   send [1] -- Send messages to other players

## Syntax
   send <race #> <message> -- This will send a message to all
                   governors of the given race.
   send <race #> <governor #> -- This will send a message to the given
                   governor of the given race.
   send block <block #> <message> -- Sends a message to every member of
                   the given block.
   send star </Star> <message> -- Send a message to all inhabitants of
                   a star.

## Description
  This command allows users to communicate with each other.  The player
must specify who they wish to contact, and a message to send to
that player.

  Race that receives your telegram gains a little knowlege of your race.
