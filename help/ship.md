# SHIP

## Name
   ship [0] -- ship report

## Syntax
   ship <#shipnum or shiptypes>

## Description
  This command allows users to survey their ships.  It is equivalent to
requesting a stock, report, and tactical report for each ship.  This command
uses scoping in the same way the other commands do.  Ship types are detailed
in the ``build'' docs.

  Additional information about ships can be obtained with the ``examine'' 
command.

## See Also
  examine, report, tactical, stats
