# STOCK

## Name
   stock [0] -- stockpile report

## Syntax
   stock <#shipnum or shiptypes>

## Description
  This command allows users to survey their ships.  It gives basic
information about cargo and crew.

  If given with no arguments, ``stock'' will start with the current scope
and give reports on all ships from there down.
