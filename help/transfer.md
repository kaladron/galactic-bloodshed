# TRANSFER

## Name
   transfer [0] -- transfer supplies to alien stockpiles

## Syntax
   transfer <race> <commodity> <amount>

## Description
   This command allows the player to transfer resources/fuel/destruct 
to another race. To do this the player must have the planet where the
transfer is to take place as the current scope. Transferred supplies
are immediately deducted from the transferring races stockpiles
and added to the targeted races stockpiles.
